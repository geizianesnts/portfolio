Jen's Kitchen

Fusion Cuisine, Website and Menu

Jen´s Kitchen restaurant website was created during a Web Design class. In this project I've learned how to link HTML pages.

Creating the website’s HTML involved the following steps:

Planning and Structure: Starting by defining the website’s purpose, target audience, and content structure. Sketching a layout that included elements like headers, navigation, main content, and footer.

Setting up the File: Using DreamWaiver to create an .html file, which served as the foundation for the webpage.

Writing HTML Structure: Beginning with a declaration to define the document type. Then, created an <html> tag to enclose the entire code, a <head> section for meta information (like title, links to CSS, and scripts), and a <body> section where the main content goes.

Adding Content: Used HTML tags to organize content <header> for the top section, <nav> for navigation, <section> and <div>for grouped content, and <footer> for the bottom. Added text, images, links, forms, etc., within these tags.

Testing and Refining: Check how the HTML renders in different browsers and adjust as needed for a responsive, user-friendly experience.
